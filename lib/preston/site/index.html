What is Preston?

Preston is a framework for building REST APIs from Mongoose models. It can be injected into an existing Mongoose application to provide a REST API, or it can create a brand new MEAN application with little effort.

Why Preston?

It's simple, extensible, and RESTful. It provides all of the functionality of a basic REST API off the bat yet allows adding new features in a very simple and sensible way.

Extensibility

Easily add authentication middleware, route-specific middlewares, or even custom queries. Preston allows you to craft your REST-compliant API the way you want it.
